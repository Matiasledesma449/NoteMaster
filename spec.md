# ESPECIFICACIÓN: Página de venta de notebooks

## 1. OBJETIVO
Página web básica, profesional y responsive (que se vea bien en celular y PC) para vender 3 notebooks.

## 2. PÚBLICO
Estudiantes de programación y profesionales creativos. Buscan rendimiento y portabilidad.

## 3. CONTENIDO ESTRUCTURA
- Header: Logo (texto "NotebookShop") + menú: Inicio, Productos, Contacto
- Hero: Frase "Potencia tu código con la notebook ideal"
- Sección productos: 3 tarjetas (cada una con: imagen, nombre, especificaciones clave, precio, botón "Comprar")
- Footer: Copyright y contacto básico (email: ventas@notebookshop.com)

## 4. RESTRICCIONES TÉCNICAS
- Tecnologías: HTML5, CSS3 (Flexbox/Grid), JavaScript VANILLA (sin frameworks)
- Diseño: colores oscuros (#0f172a) con acentos en azul eléctrico (#3b82f6)
- Debe ser responsive: en móvil, las tarjetas se apilan una abajo de otra.
- No requiere backend ni base de datos real. Los botones "Comprar" solo muestran alerta: "🚀 Simulación de compra - Envíanos un email a ventas@notebookshop.com"
